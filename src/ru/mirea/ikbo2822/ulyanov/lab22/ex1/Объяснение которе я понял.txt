Это патерн проектирования, который используется для создания объектов
без явного указания конкретных классов создаваемых объектов.
Он представляет собой интерфейс для создания
экзембляров класса , но позволяет подклассам выбирать класс для создания:

Абстрактная фабрика: методы для создания объекта
Конкретная фабрика : создает экземпляры конкретных классов
Продукты: Классы, объекты которых создаются фабрикой
